Write JAVA program, which reads n and n numbers and finds how many of them are prime. e.g. if n=6 and given numbers are 7, 13, 16, 27, 11 and 9 then answer is 3 since the prime numbers are 7, 13 and 11. (Hint: Smallest factor of a prime number is equal to itself). [Smallest factor]


public static void prime() {
	Scanner sc = new Scanner(System.in);
	int n = 6;
	int counter = 0;
	int totle = 0;
	
	for (int i = 0; i < n; i++) {
	    int num = sc.nextInt();
	    
	    for (int j = 2; j <= num; j++) {
	        if (num % j == 0) { counter++; }
	    }
	    
	    if (counter == 1) {
	        totle++;
	    }
	
	    counter = 0;
	}
	System.out.print(totle);
}
